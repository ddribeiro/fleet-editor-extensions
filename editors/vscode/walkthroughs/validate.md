## Real-Time Validation

The Fleet GitOps extension validates your YAML as you type:

- **Red underlines** — Errors: unknown keys, invalid types, missing required fields
- **Yellow underlines** — Warnings: deprecated keys, query syntax issues
- **Strikethrough** — Deprecated keys that will be removed in a future Fleet version

### Quick Fixes

Click the lightbulb (or press `Ctrl+.` / `Cmd+.`) to see available fixes:

- Fix typos in key names
- Replace deprecated keys with their new equivalents
- Add missing required fields

### Hover Documentation

Hover over any key to see its description, valid values, and examples.
